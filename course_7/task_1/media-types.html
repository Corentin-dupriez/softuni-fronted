Media Types

https://developer.mozilla.org/en-US/docs/Web/CSS/@media#media_types

all

Suitable for all devices

@media all {
  * {
    padding: 0;
    margin: 0;
    box-sizing: border-box;
  }

  body {
    padding: 2em;
    font: 14px/1.5 Lato, sans-serif;
  }    

  p {
    padding-bottom: 1.5em;
  }
}
